字符串格式验证类
======================

## 批量验证

```
$rules = [
    'user_name' => [
        // 验证方法只有一个参数
        'isNotEmpty'   => '请输入用户名', 
        'isSafeString' => '用户名只允许输入字母、数字和下划线',

        // 验证方法需要多个参数
        'minLen'       => ['msg' => '用户名不能小于3个字符', 'minLen' => 3],
        'maxLen'       => ['msg' => '用户名不能超过20个字符', 'maxLen' => 20],
    ],
    'email'    => [
        'isNotEmpty'   => '请输入邮箱', 
        'email'        => '邮箱格式错误',
    ]
]

```
